# Advent of Code 2024

Solving Advent of Code puzzles in Ruby!

## Tests

To run all tests:

```
rake test
```

To run a single test file:

```
ruby -Itest tests/advent_of_code_2024/test_day1.rb
```

To run a single test:

```
ruby -Itest tests/test_advent_of_code_2024.rb --name test_day1_answers
```
